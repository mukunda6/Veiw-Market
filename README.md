# 📊 View Market – Smart Market Price Viewer

## 🚀 Project Overview
**View Market** is a web-based application that allows users to view daily market prices of essential commodities such as Vegetables, Meat, Fuels, and Metals across multiple cities.

The platform helps users easily compare prices based on location and category through a simple and user-friendly interface.

---

## 🎯 Features
- ✅ View market prices city-wise
- ✅ Categories available:
  - 🥦 Vegetables
  - 🍖 Meat & Seafood
  - ⛽ Fuels
  - 🪙 Metals
- ✅ Multiple city support:
  - Hyderabad
  - Chennai
  - Delhi
  - Mumbai
  - Vizag
- ✅ Simple navigation system
- ✅ Responsive UI design
- ✅ Login & Help pages included

---

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3
- **Design:** Static Web UI
- **Platform:** Web Browser
